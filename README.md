# TikTok Hook Generator ⚡

Generate 10 high-converting TikTok video hooks in seconds — powered by AI.

## What it does

The hook is the single highest-leverage variable in TikTok Shop performance.
It determines whether a viewer watches or scrolls in the first 3 seconds.

Fill in your product details, pick a tone, and get 10 ready-to-use hooks
instantly. Copy the one you like and start filming.

**Input:**
- Product name
- Product category
- Target audience
- Tone (Funny / Educational / Emotional / Trendy / Shocking)

**Output:**
- 10 high-converting TikTok hooks
- One-click copy for each hook

## Why I built this

Built while working on [SpreadableAI](YOUR_LINK_HERE) — an AI operating
system for TikTok Shop agencies.

Hooks are what separate a 100-view video from a 1M-view video. We wanted
a fast tool that any creator in our network could use before hitting record.
